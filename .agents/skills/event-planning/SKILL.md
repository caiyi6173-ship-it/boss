# 活动策划全流程技能 (Event Planning Skill)

本技能用于指导 Agent 进行线下活动（如商场活动、音乐节宣传、品牌发布会）与线上联动营销的全流程策划与执行。

## 适用场景
- 商场/线下品牌宣传活动
- 音乐节/演出/展览推介活动
- 小红书 / 抖音短视频联动宣发
- 活动 SOP 流程制定与现场风控

---

## 阶段一：活动前策划与筹备 (Pre-Event)

### 1.1 活动定位与目标
- **目标设定**：明确活动 KPI（如现场人流量、小红书/抖音曝光量、转化率等）。
- **活动主题**：设计引人入胜的活动标语、主视觉风格与氛围基调。

### 1.2 内容与媒体策略
- **物料清单**：背景板、宣传海报、现场音响、周边赠品、录播设备。
- **媒体渠道规划**：
  - **小红书**：侧重精美图文、现场打卡攻略、优质话题 Tag、打卡领周边。
  - **抖音**：侧重 15-30s 爆点短视频、现场音乐/表演高光切片、挑战赛与原声使用。

### 1.3 关键时间节点排期 (Milestones)
- T-7 天：策划方案终审，确定预算与采购清单。
- T-3 天：拍摄设备调试、小红书/抖音预热文案与视频脚本定稿。
- T-1 天：现场搭建、设备彩排、人员分工与应急方案演练。

---

## 阶段二：活动现场执行 SOP (Event Day Execution)

### 2.1 现场人员职责表
| 角色 | 主要职责 |
|------|----------|
| 活动总指挥 | 整体控场、时间节点调度、突发事件处理 |
| 摄影/摄像师 | 现场高光时刻捕捉、短视频素材拍摄 |
| 新媒体运营 | 实时素材整理、现场文案撰写、小红书/抖音即时上传 |
| 现场引导员 | 人流疏导、打卡互动指引、礼品发放 |

### 2.2 风险防范与应急措施 (Risk Mitigation)
- **设备故障**：备用麦克风、充电宝、存储卡、备用相机。
- **网络异常**：配备移动 Wi-Fi 保证上传顺畅。
- **审核风控**：小红书/抖音发布内容提前进行合规与敏感词自查。

---

## 阶段三：活动后复盘与资产沉淀 (Post-Event)

### 3.1 数据汇总与汇报
- 汇总小红书播放/点赞量、抖音曝光量、现场参与人数。
- 导出阶段性进度与结果简报。

### 3.2 资产沉淀
- 归档视频原片、高精图库。
- 将本次成功的执行步骤整理沉淀为可复用的 SOP 模板。
