  Boss Task Agent 使用说明
  老板语音 → 结构化任务 / SOP / 看板 / 复盘
═══════════════════════════════════════════════════════════════

【第一步：填写 API Key（仅首次）】

1. 用记事本打开本目录下的 .env 文件
2. 找到这一行：
     OPENAI_API_KEY=your-api-key-here
3. 把 your-api-key-here 替换为你的真实密钥
   （密钥由管理员提供，格式类似 sk-xxxxxxxx）
4. 保存并关闭

   提示：OPENAI_BASE_URL 和 OPENAI_MODEL 一般不需要改，
   除非管理员另有指示。


【第二步：准备语音文件】

方式 A：使用影刀 RPA 自动生成

1. 打开影刀 Workbook 生成工具：
   https://api.winrobot360.com/redirect/robot/share?inviteKey=c7b51ef45fe9af10
2. 按影刀流程执行后，生成项目需要的 Excel 文件
3. 将生成结果保存为：
     input\Workbook.xlsx

方式 B：手动准备 Excel

1. 将老板的语音转录 Excel 文件放入 input 文件夹
2. 文件名必须是 Workbook.xlsx
   （或在程序界面点击「浏览」选择其他 Excel 文件）


【第三步：运行程序】

1. 双击 BossTaskAgent.exe
2. 程序窗口打开后：
   - 点击「选择语音文件」可以选择不同的 Excel 文件
   - 点击「一键全流程」开始处理（约 10-30 秒）
   - 处理完成后会自动弹出看板网页
3. 也可以单独执行某个步骤：
   - 「解析任务」：仅将语音解析为结构化任务
   - 「生成 SOP」：生成标准操作流程模板
   - 「导出表格」：生成 Excel 任务表格
   - 「生成看板」：生成可视化 HTML 看板
   - 「生成复盘」：生成复盘报告


【输出文件说明】

  output/structured_tasks.json   结构化任务数据
  output/task_table.xlsx         Excel 任务表格
  output/task_dashboard.html     可视化看板（浏览器打开）
  output/retrospective_report.md 复盘报告
  sop_templates/                 SOP 流程模板


【常见问题】

Q: 点击按钮后很久没反应？
A: AI 分析需要 10-30 秒，请耐心等待日志区出现结果。
   如果超过 60 秒无反应，检查网络连接和 API Key 是否正确。

Q: 提示「API Key 未设置」？
A: 请检查 .env 文件是否正确填写了密钥。
   也可以直接在程序顶部的输入框中填写并点击「保存」。

Q: 程序闪退？
A: 尝试右键 → 以管理员身份运行。
   如仍有问题，请将 output 目录下的日志发给技术支持。
